About

A simple photo browser.
Website

Building on Linux (Ubuntu 13.04 used)

Required applications:

cmake

Required libraries:

boost
gtkmm
gtkglext
xdg-basedir

Steps:

git clone git://github.com/pilkch/library.git
git clone git://github.com/pilkch/diesel.git
cd diesel/project/
cmake .
make
cd ..
./diesel
